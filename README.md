# Тестовое скб-контур

## Задание
Написать бота для игры, которую мы назвали «Морская торговля».

В игре вы играете за торговый корабль, задача которого заработать как можно больше денег, продавая разные товары в разные города.

Бот, которого вы напишете, будет соревноваться по количеству заработанных денег с ботами других участников. Следить за своими результатами можно на [лидерборде](https://ulearn.me/Analytics/RatingByPoints?courseId=frontend-internship-2020&slideId=eee4d832-d5fc-4563-aeab-b219f2e98d26)

## Запуск игры
1. Распакуй архив с задачей
2. Установи все зависимости в директории задачи при помощи команды в терминале npm install
3. Запусти приложение, выполнив команду npm start
4. Перейди в браузере по адресу localhost:3000
